Money Tracker — это одностраничное веб-приложение для учёта личных расходов. Приложение позволяет добавлять, редактировать и удалять записи о тратах, а также фильтровать и сортировать их по различным критериям. Данные сохраняются в локальном хранилище браузера, поэтому они не теряются после перезагрузки страницы.

Функционал:
1. Добавление траты — указание названия, суммы и категории
2. Редактирование траты — изменение любых полей существующей записи
3. Удаление траты — удаление с подтверждением
4. Фильтрация по категориям — отображение только выбранной категории
5. Сортировка — по дате (новые/старые) и по сумме (дорогие/дешёвые)
6. Общая сумма — автоматический подсчёт всех отображаемых трат
7. Drag-and-drop — изменение порядка записей перетаскиванием мышью
8. Сохранение данных — автоматическое сохранение в localStorage
